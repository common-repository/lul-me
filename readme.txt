== Description ==

**Lock your content/youtube video/etc behind paywall. Reveal content after user payment.**

Let your audience reveal part of your blog post content OR youtube video (unlisted) OR Anything by paying you a price set by you during post publish.

**Hiding content behind paywall is very simple using shortcode:**

This is free content for my audience

**[ppv price="5"]**

This is paid content for my audience, visible only after purchase.

**[/ppv]**

**--What you can hide behind paywall:--**

- Answer to solve a user question (Paid answers)

- Part of your blog post content.

- Youtube video (unlisted)

- Code snippets

- Google drive / Dropbox / One drive: file links (aka download links)

..........etc
